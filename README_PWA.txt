Stock et Inventaire — Pack PWA (GitHub Pages)

FICHIERS :
- bon_retour_stock_app.html  (app)
- manifest.webmanifest       (install)
- sw.js                      (offline)
- icon-192.png
- icon-512.png

IMPORTANT :
Pour GitHub Pages, le fichier d'accueil doit s'appeler index.html.
=> Renomme bon_retour_stock_app.html en index.html (ou uploade une copie sous ce nom).

ÉTAPES (résumé) :
1) Dézippe.
2) Upload tous les fichiers à la racine du dépôt GitHub.
3) Renomme bon_retour_stock_app.html -> index.html.
4) Settings -> Pages -> Deploy from a branch -> main -> /(root) -> Save.
5) Ouvre l'URL GitHub Pages de ton dépôt et installe via Chrome.
